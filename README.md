# QQBackUp

# QQ聊天记录备份助手

![](https://images2018.cnblogs.com/blog/1192699/201803/1192699-20180303201019669-8052439.png)

####  项目地址
https://github.com/guoyaohua/QQBackUp

####  开发环境

Netbean

JAVA

#### 功能描述

1. 自动搜索硬盘中的QQ聊天记录文件
2. 可选择的备份QQ数据记录
3. 还原QQ数据记录
